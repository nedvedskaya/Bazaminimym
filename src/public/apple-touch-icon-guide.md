# Apple Touch Icon Setup

## Что это?
Apple Touch Icon - это иконка, которая отображается когда пользователь добавляет сайт на домашний экран iPhone/iPad.

## Требования
- Размер: 180x180 пикселей (для Retina дисплеев)
- Формат: PNG
- Название файла: `apple-touch-icon.png`
- Без прозрачности (должен быть фон)

## Как создать

### Вариант 1: Онлайн генератор
1. Откройте https://favicon.io/
2. Загрузите ваш логотип или используйте текст "БМ"
3. Выберите:
   - Цвет фона: #e3ee6b
   - Цвет текста: #0A0A0A
   - Шрифт: Arial Black или другой жирный
4. Скачайте apple-touch-icon.png (180x180)
5. Поместите в `/public/apple-touch-icon.png`

### Вариант 2: Figma/Photoshop
1. Создайте артборд 180x180px
2. Фон: #e3ee6b
3. Текст: "БМ", цвет #0A0A0A, размер ~80-100px, выровнять по центру
4. Экспортируйте как PNG
5. Сохраните как `/public/apple-touch-icon.png`

### Вариант 3: ImageMagick (командная строка)
```bash
# Конвертация SVG в PNG 180x180
convert favicon.svg -resize 180x180 -background "#e3ee6b" -flatten apple-touch-icon.png
```

## Использование

После создания файла, добавьте в `index.html` (уже добавлено):
```html
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
```

## Дополнительные размеры (опционально)

Для полной совместимости можно создать несколько размеров:
- `apple-touch-icon-152x152.png` - iPad Retina
- `apple-touch-icon-167x167.png` - iPad Pro
- `apple-touch-icon-180x180.png` - iPhone (обязательно)

## Проверка

1. Откройте сайт на iPhone Safari
2. Нажмите кнопку "Поделиться"
3. Выберите "На экран Домой"
4. Проверьте что иконка отображается правильно
