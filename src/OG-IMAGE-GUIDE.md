# Руководство по созданию Open Graph изображения

## Требования к Open Graph изображению

### Размеры и формат
- **Рекомендуемый размер**: 1200x630 пикселей
- **Минимальный размер**: 600x315 пикселей
- **Соотношение сторон**: 1.91:1
- **Формат**: JPG или PNG
- **Максимальный размер файла**: до 8 MB (рекомендуется до 1 MB)

### Дизайн

Создайте изображение со следующими элементами:

1. **Фон**: Черный (#0A0A0A) или градиент от черного к темно-серому
2. **Акцентный цвет**: Желто-зеленый (#e3ee6b)
3. **Текст**:
   ```
   БАЗОВЫЙ МИНИМУМ
   Курс по продажам для автосервисов
   5 модулей | 2490₽ | Старт 12 января
   ```
4. **Визуальные элементы**: Металлические текстуры или абстрактные элементы
5. **Логотип/иконка**: Если есть
6. **Безопасная зона**: Оставьте отступы 40px со всех сторон

### Инструменты для создания

1. **Canva** (онлайн, бесплатно):
   - Шаблон: Social Media → Facebook Post
   - Измените размер на 1200x630px

2. **Figma** (бесплатно):
   - Создайте frame 1200x630px
   - Экспортируйте как PNG или JPG

3. **Adobe Photoshop/Illustrator**:
   - Создайте документ 1200x630px
   - DPI: 72
   - Цветовой режим: RGB

### Размещение файла

1. Сохраните изображение как `og-image.jpg` или `og-image.png`
2. Поместите в папку `/public/`
3. Обновите `index.html`:

```html
<!-- Open Graph / Social Media Meta Tags -->
<meta property="og:image" content="https://bazovyminimum.ru/og-image.jpg">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:image:alt" content="Базовый Минимум - Курс по продажам для автосервисов">

<!-- Twitter -->
<meta name="twitter:image" content="https://bazovyminimum.ru/og-image.jpg">
```

### Проверка

После добавления изображения проверьте его отображение:

1. **Facebook Debugger**: https://developers.facebook.com/tools/debug/
2. **Twitter Card Validator**: https://cards-dev.twitter.com/validator
3. **LinkedIn Post Inspector**: https://www.linkedin.com/post-inspector/
4. **Telegram**: Отправьте ссылку в Saved Messages

### Цветовая схема

- Основной фон: `#0A0A0A` (черный)
- Акцент: `#e3ee6b` (желто-зеленый)
- Текст заголовка: `#FFFFFF` (белый)
- Текст подзаголовка: `#FFFFFF` с прозрачностью 80%
- Декоративные элементы: Градиент от `#e3ee6b` до `#c5d060`

### Шрифты

- Заголовок: System font stack (жирный, большой)
- Подзаголовок: System font stack (средний)
- Детали: System font stack (обычный, uppercase)

### Контрольный список

- [ ] Размер 1200x630px
- [ ] Формат JPG или PNG
- [ ] Размер файла < 1 MB
- [ ] Текст читаемый и контрастный
- [ ] Отступы от краев минимум 40px
- [ ] Соответствует брендингу сайта
- [ ] Протестировано в Social Media Debuggers
- [ ] Добавлено в index.html

## После создания

Обновите следующие файлы:

1. `/index.html` - добавьте meta теги
2. `/public/manifest.json` - добавьте icons
3. `/SEO-GUIDE.md` - отметьте как выполненное
